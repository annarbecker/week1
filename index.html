About Me

As a Portland native, I grew up in the city and after high school graduation was ready for a change of scenery. In fall 2008, I found myself in Gambier, Ohio as a freshman at Kenyon College. During my time at Kenyon I received a BA in Sociology with a minor in French.

After graduation, and four years of miserable Ohio winters, I moved back to Portland. As a recent grad I was unsure of exactly what I wanted to do. I began volunteering at a local nonprofit located in the St. Johns neighborhood, Portland After-School Tennis & Education (PAST&E). After my first day of volunteering I was extremely impressed with the organization's mission, impact, and staff. I continued to volunteer throughout the summer of 2012 and in late August of that year the Executive Director offered me a job.

I worked at PAST&E for three and a half years. In my time with the organization I was a Coach/Tutor and provided one-on-one tutoring and tennis instruction to elementary through high school aged children. In the second year of my employment I took over as Manager of St. Johns Racquet Center, the facility in which the organization is located, and oversaw day-to-day operations for public use of the facility's courts. Most recently, I worked as the Development Director writing grants, cultivating donors, and organizing major fundraisers.

Throughout my schooling and employment I always enjoyed the design and creative elements of my work the most. During my time at PAST&E I had the opportunity to create promotional materials in Adobe Illustrator and redesign the website using Square space. As I started to look into what I wanted to do next in my career path I knew that I wanted it to include design and technology.

Right around this time I was hearing more and more about code boot camps. I was interested in learning more about the schools and speaking with a graduate. I reached out to a former colleague of mine, Maureen ("Moe") Dugan. When I joined the PAST&E team Moe was the Development Director and I worked closely with her before she left the organization in winter 2013. I sat down with Moe and flooded her with questions about her experience with coding and as a student at Epicurus. Moe was very encouraging in my exploration of the field. After our meeting I decided I knew what I wanted to do next and submitted my Epicurus application.

I am excited to begin my programming journey and eager to see where it takes me!

Projects

Below is a list of projects I created in my first week at Epicodus!

  • Cookie Recipe

    • The project displays a Best Chocolate Chip Cookies recipe including a brief history of the cookie, an image, and complete lists of ingredients and directions. The project demonstrates the use of block and inline elements. The project was built and styled with HTML.

  • First Webpage

    • The project displays Anna and Izzy's Favorite Things, which are categorized by food and hobbies with the use of ordered and unordered lists. Images are and links are included to better depict our favorite foods. The project demonstrates the use of block and inline elements with the addition of CSS for text styling. The project was built and styled with HTML and CSS.

  • Favorite Places

    • The project displays Anna and Izzy's Favorite Places, which are categorized by Anna and Izzy with the use of unordered lists. The page also has a link to the previous project page, "Our Favorite Things." The project demonstrates the use of block and inline elements with the addition of CSS for text styling. The project was built and styled with HTML and CSS.

  • Island

    • The project displays information about a Private Island Resort. An image and links are included to provide more information about the island. The project demonstrates the use of block and inline elements with the addition of CSS for page and text styling. The project was built and styled with HTML and CSS. This project used branching to create branches for different styling of the same index.html page.

  • Cupcake

    • The project displays information about Portland's Best Cupcakes. Important headers and paragraphs are highlighted to stand out. Images and a link are included to provide more information about the cupcakes. The project demonstrates the use of block and inline elements and uses CSS classes to style multiple elements on the page. The project was built and styled with HTML and CSS.

  • Boring

    • The project displays information about a boring class subject, Ethnolichenology. The project describes Ethnolichenology, the study of lichens, provides an image of lichen, and follows with additional information on the subject. The project demonstrates the use of block and inline elements and uses div to style sections on the page. The project was built and styled with HTML and CSS.

  • Test Project

    • The project displays a blog with columns for entries on days 1, 2, and 3. The project demonstrates the use of block and inline elements, uses div to style sections on the page, and uses floats to create separate columns for the blog entries. The project was built and styled with HTML and CSS.

  • Animal

    • The project displays page for an animal shelter. The page shows 5 animals available to be adopted, including pictures and descriptions of each animal. The project demonstrates the use of block and inline elements, uses div and span to style sections and elements on the page, uses floats to create separate columns for the animal profiles, uses the box model to adjust padding, margins, and border size, the project also styles elements nested within the profiles using the idea of cascading. Media queries added to the project change the page's background color and column width when the page width is less than or equal to 480px. The project was built and styled with HTML and CSS.

  • Vaction

    • The project displays popular vacation destinations with information about each location. The project demonstrates the use of block and inline elements, uses div and span to style sections and elements on the page using Bootstrap classes, uses the box model to adjust padding, margins, and border size, the project also styles elements nested within the profiles using the idea of cascading. The project was built and styled with HTML, CSS, and Bootstrap. JavaScript was used for minor site animations in the carousel and navigation bar.
